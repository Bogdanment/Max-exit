# 🚀 Deployment Documentation

This folder contains all deployment guides and scripts for Exit Accessories.

## 📁 Structure

### 📚 Guides (выбирайте платформу)
- **DIGITALOCEAN_DEPLOY.md** - Подробное руководство для DigitalOcean VPS
- **DEPLOYMENT.md** - Общее руководство по деплою
- **HEROKU_DEPLOY.md** - Деплой на Heroku
- **RENDER_DEPLOY.md** - Деплой на Render.com
- **QUICK_DEPLOY.md** - Быстрые варианты деплоя

### 🔧 Scripts (для автоматического деплоя)
- **deploy-digitalocean.ps1** - Windows PowerShell скрипт для DigitalOcean
- **deploy-digitalocean.sh** - Bash скрипт для VPS
- **start.sh** - Linux стартовый скрипт
- **start.bat** - Windows стартовый скрипт

### ⚙️ Configuration Files
- **Procfile** - Heroku конфигурация
- **railway.json** - Railway конфигурация
- **render.yaml** - Render конфигурация
- **vercel.json** - Vercel конфигурация
- **netlify.toml** - Netlify конфигурация
