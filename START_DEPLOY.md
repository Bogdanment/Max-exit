## ❓ FAQ

**Q: Нужно ли что-то менять в коде прежде чем запустить скрипт?**  
A: Нет! Скрипт всё настроит автоматически.

**Q: Что если SSH не подключается?**  
A: Проверьте что VPS включен, у вас правильный пароль, и IP адрес корректный.

**Q: Как часто нужно обновлять приложение?**  
A: Просто запустите `git pull && docker-compose... up -d --build` на VPS.

**Q: Где находятся логи приложения?**  
A: Используйте `docker-compose logs -f` чтобы смотреть логи в реальном времени.

**Q: Как включить SSL/HTTPS?**  
A: Используйте `certbot certonly --nginx -d your-domain.com` на VPS (если у вас есть домен).

**Q: Можно ли масштабировать приложение?**  
A: Да, для больших нагрузок можно добавить LoadBalancer, Redis кеш, или Kubernetes позже.
